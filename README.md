# Índice de Autenticidad Metodológica (IAM)

Este repositorio contiene el código fuente del software **IAM**, desarrollado como parte de la investigación: *"El límite de la conciencia en el consumo: Modelo híbrido de validación de encuestas de marketing"*.

## Descripción
El IAM es una herramienta algorítmica desarrollada en Python que audita la validez conductual de estudios de mercado. Permite contrastar datos declarativos (encuestas) con datos conductuales reales, penalizando la presencia de sesgos cognitivos no controlados.

## Requisitos
* Python 3.8+
* Streamlit
* Pandas
* Numpy

## Cómo ejecutar localmente
1. Clona este repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta la aplicación: `streamlit run iam_index.py`
